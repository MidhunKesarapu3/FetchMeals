# FetchMeals IOS APP

## Description
- Application fetches the meal information and display them as list.
- Each meal contains the prepartion method and ingeredients.

## Requirements
-  Download and Install XCode latest version from [here](https://apps.apple.com/us/app/xcode/id497799835?mt=12).

## Run Application

1. Clone the project from [here](https://github.com/MidhunKesarapu3/FetchMeals.git).
2. Open the project from XCode.
3. Click on Build icon from XCode to open the application on Emulator.
